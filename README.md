**Face Recognition Attendance System**

An automated attendance management solution that leverages Computer Vision and Deep Learning to identify individuals and log attendance in real-time. This system eliminates manual errors, prevents "proxy" attendance, and provides a seamless digital experience for educational institutions and corporate offices.

**Key Features**

**Real-time Detection:** High-speed face detection using Haar Cascades or HOG.

**Biometric Accuracy:** Advanced feature extraction for precise individual recognition.

**Automated Logging:** Instant attendance marking in .csv or database formats with timestamps.

**Intuitive GUI:** User-friendly interface built with JavaFX for easy registration and tracking.

**Reporting:** Generates daily/monthly attendance reports automatically.

**Graphical User Interface (GUI)**

The system features a streamlined dashboard for administrators and users.

1. Main Dashboard
   A central hub for starting the camera, registering new students, and viewing records.

2. Live Recognition Window
   Displays the live webcam feed with bounding boxes and names of recognized individuals.


**Tech Stack**

Language: Java

Computer Vision: OpenCV

GUI Framework: JavaFx

Data Handling: MySQL
